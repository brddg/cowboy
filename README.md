# Cowboy

App used to return 500 character chunks of valid email addresses.

https://rma-cowboy-app.herokuapp.com/

## setup
```
npm install
```

## watch and autobuild
```
npm run watch
```

## run server
```
npm start
```

## manual build
* create bundle.js
```
npm run build
```

## test (with Jest)

```
npm test
```

## lint

* using ESLint

```
npm run lint
```
